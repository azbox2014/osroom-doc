## 介绍

OSROOM是基于Python3 Flask开发的开源Web系统, 也是一个CMS(内容管理系统)
可以用来建立企业官网, 个人博客网站, 提供Rest Api请求数据可以作为微信小程序的服务端

#### &nbsp;&nbsp;运行平台
可以运行于Linux系统, Windows未测试

#### &nbsp;&nbsp;版本
目前最新版本: 1.0 Beta
